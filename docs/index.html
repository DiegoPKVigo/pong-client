pong

Pong client

Pong is a HTML5 game based on the classic pong game. This project is written in TypeScript and designed with Canvas.

Requirements

In order to be able to deploy Pong you need a web server and Node.js installed.

Building

Follow the next steps to build the project:

  1. Clone the repository
  2. Change to the Pong directory: cd pong-client
  3. Install the required dependencies: npm install
  4. Compile and bundle TypeScript sources with Webpack: npm run webpack

Usage

Link the generated JavaScript file in dist directory to a web page and deploy both files in your hosting or web server.
